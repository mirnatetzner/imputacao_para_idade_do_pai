# Datasus - SINASC - Fecundidade_masculina
Código utilizado na dissertação "Fecundidade Masculina no Brasil e fontes de dados: Potencialidades para
além das limitações metodológicas", para a obtenção do titulo de mestre na ENCE.

Estima-se os indicadores TFT e TEFs masculinas para 2022 a partir de métodos de imputação para dados faltantes.
